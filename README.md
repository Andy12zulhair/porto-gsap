# Andi Zulhair Portfolio — GSAP & Vite

A high-end, interactive portfolio built with **GSAP**, **Vite**, and **Lenis Scroll**. Features a luxury aesthetic with smooth transitions, magnetic interactions, and a refined grain overlay.

## Tech Stack
-   **Animations**: GSAP (Timeline, Draggable)
-   **Scrolling**: Lenis Smooth Scroll
-   **Bundler**: Vite
-   **Texture**: Grained.js
-   **Styling**: Pure CSS

## Getting Started
1.  `npm install`
2.  `npm run dev`
